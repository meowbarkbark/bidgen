# 원가계산서 자동검증 (Cost Validator Frontend MVP)

React + TypeScript + Vite 기반의 원가계산서 자동검증 웹앱 프론트엔드 MVP입니다.

## 실행 방법

```bash
pnpm install
pnpm dev      # 빌드 후 http://127.0.0.1:4173 에서 미리보기
```

## 스크립트

- `pnpm build` — 타입 체크 후 프로덕션 빌드
- `pnpm preview` — 빌드 결과 미리보기
- `pnpm test` — 타입 체크 (`tsc --noEmit`)

## 구조

- `src/` — 애플리케이션 소스 (컴포넌트, 화면, 데이터, 유틸)
- `frontend/` — 프론트엔드 관련 리소스
- `docs/` — 문서
- `qa/` — QA 스크린샷
